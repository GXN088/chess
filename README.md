Пояснение:
Этот псевдокод описывает структуру программы, которая создает иерархию классов для шахматных фигур. Класс ChessPieces (ШахматнаяФигура) является родительским для всех конкретных фигур: короля, ферзя, ладьи, коня, слона и пешки. Метод main в данный момент не содержит логики и служит только точкой входа.
